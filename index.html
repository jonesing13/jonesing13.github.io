Minneapolis Skyway Navigation Mobile App

Project Brief

Project Hook

Personal reasoning and story/framing, including:

  • Reinvigorate Downtown Minneapolis (reference things; quotes, etc.)
  • Build community opportunities (more reliant on personal anecdotes, having moved downtown in 2023)
  • Support local businesses (reference things; quotes, etc.)

The Plan:

  1. Evaluate similar tools (like Google Maps, Apple Maps, Skywayaccess.com, Skyway.run
  2. Draft low-fi mock ups of a Skyway Mobile App
  3. Write a Research Plan, and get feedback from Experts (Brittany Schiesel)
  4. Build a prototype (or two) for user feedback
  5. Perform user interviews and usability tests
  6. Analyze user feedback and outline design recommendations for the Mobile App
  7. Create a Basic Style Guide for the platform, including colors, typography, and naming
  8. Identify future research opportunities and potential phases for deliverables

The Process:

  1. Evaluate existing tools
  2. Sketches/mock ups of a Skyway Mobile App
  3. Research Planning, drafting a script, etc. (with Brittany Schiesel)
  4. Build a prototype (or two) for user feedback (struggled w navigation; illustrating all that? only showing 3-5 steps in the route? what verbiage is going to resonate?)
  5. Perform user interviews and usability tests
  6. Analyze user feedback and outline design recommendations for the Mobile App
  7. Create a Basic Style Guide for the platform, including colors, typography, and naming
  8. Identify future research opportunities and potential phases for deliverables (based off user feedback, needs for delivery, etc.)
